# xamarin-todo-with-realm




Xamarin.Formsの [Todoサンプル](https://developer.xamarin.com/samples/xamarin-forms/Todo/) をSQLite仕様から [Realm Xamarin](https://realm.io/docs/xamarin/latest/) 仕様に置き換えたもの。
